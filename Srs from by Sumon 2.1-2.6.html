2.1 Product Perspective

It is a very useful app for those who are seeking for date online. Although there are some similar apps available on the internet but these are expensive to use and require costly subscriptions. It is free to use and you just have to download free app for app store or visit web app through your browser.

2.2 Product Functions

The Product will mainly provide these functions to its users:-

user:

  • Register/Signup
  • Login
  • Displaying swipeable cards
  • Create/Edit Profile
  • Set Likes/Love
  • Create matches
  • Chat

Admin:

  • An admin can add , edit or delete a user account.
  • An admin can approve or dicard a user review

2.3 User Classes and Characteristics

This app is for all the users especially for the people aged between 17-40. Anyone with basic computer skills and internet skill can use this app because of its easy-to-use UI.

2.4 Operating Environment

  • Operating environment is as listed below.

    There will be two main operating environments:-

    • Web browsers
    • Android : Must support Android 4.4 and up versions

2.5 Design and Implementation Constraints

Internet connection is required. Without an internet connection, the users cannot log in to the system. So, they cannot interact with the system without internet.

Another constraint is data capacity. As we are not using our own data center, we are might face issue with capacity.
